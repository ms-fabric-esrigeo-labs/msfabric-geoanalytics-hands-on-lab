# Lab Series Overview

**Geospatial Analytics with Microsoft Fabric and Esri: From Data Ingestion to Insightful Visualization**

**Estimated duration:** ~4 hours

This hands-on lab series introduces large-scale spatial analytics using
Microsoft Fabric, PySpark, and Esri GeoAnalytics.

Participants learn how to:
- Configure and authenticate GeoAnalytics in Fabric
- Prepare and engineer spatial datasets
- Perform spatial aggregation, clustering, and hotspot analysis
- Analyze spatial and temporal trends
- Visualize results using geospatial plotting capabilities
