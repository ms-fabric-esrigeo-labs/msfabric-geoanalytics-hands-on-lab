
# msfabric-geoanalytics-hands-on-lab
Hands-on lab exploring large-scale geospatial analytics with Microsoft Fabric and Esri. Learn to ingest, prepare, analyze, and visualize spatial data using PySpark and GeoAnalytics—covering geometry creation, spatial operations, aggregation, clustering, and hotspot analysis to drive location‑aware insights in the cloud.
> ⚠️ **Authoritative Lab Guide**
> The step-by-step instructions for this hands-on lab are delivered via CloudLabs.
> This repository provides context, structure, and supporting assets only
